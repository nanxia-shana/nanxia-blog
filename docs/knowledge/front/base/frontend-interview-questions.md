# 前端面试题解析

## HTML

### 1. 对HTML语义化的理解

#### 1-1. 关键词

> - **HTML 语义化**

#### 1-2. 考察点

::: info 考察点

- **基础概念**：在编写 HTML 时，根据内容的结构和意义，选择最恰当、具有明确含义的标签，让代码“自解释”，既方便人阅读，也方便机器（浏览器、搜索引擎、屏幕阅读器）理解。
- **标签选择**：是否熟悉常用语义标签及场景（header、nav、main、article、section、aside、footer、h1~h6 等）
- **SEO 搜索引擎优化**：搜索引擎爬虫会分析标签含义，语义化结构能帮助搜索引擎爬虫理解页面主题与层次
- **代码可读性与可维护性**：语义化能提升代码可读性，让团队成员快速理解结构。
- **遵循 Web 标准**：HTML 的结构与表现分离原则：结构（HTML）表达“这是什么”；样式（CSS）表达“长什么样”；行为（JS）表达“做什么”。
:::

#### 1-3. 回答

::: tip 回答：
HTML中的部分标签有着独特的含义，而语义化就是指，在写页面内容时选用合适的标签。通俗来讲就是用正确的标签做正确的事情。
因为选用了带有语义化的标签去书写代码，所以HTML语义化增强了代码的可读性，使代码结构更加清晰，并且十分有利于SEO。
:::

### 21. JavaScript（JS）的数据类型有哪些？

#### 21-1. 关键词

> - **数据类型**

#### 21-2. 考察点

::: info 考察点

- **基础概念**：是否清楚 JavaScript 中的数据类型分类和数量。
- **基本数据类型**：是否能列举出 JavaScript 中的基本数据类型（Undefined、Null、Boolean、Number、String、Symbol、BigInt）。
- **引用数据类型**：是否能列举出 JavaScript 中的引用数据类型（Object、Array、Function、Date、RegExp 等）。
- **类型判断**：是否了解如何使用 `typeof`、`instanceof` 等方法判断数据类型。
- **类型转换**：是否了解 JavaScript 中的隐式和显式类型转换。
:::

#### 21-3. 回答

::: tip 回答：
JS中一共有八种数据类型，然后这八种数据类型又可以分为原始数据类型和应用数据类型，其中原始数据类型有七种：

- 第一种是 <span class="Red-Font">**undefined**</span> 类型，它只有一个值是 undefined ，代表未定义。如果我们定义一个变量但是不给他赋值的话，那么它当前的值就是 undefined；
- 第二种类型是 <span class="Red-Font">**null**</span> 类型，他也只有一个值是 null，代表空指针对象。一个变量如果后面会被用来赋值作为对象的话，那么我们可以在定义变量的时侯就把它赋值为 null 作为初始化；
- 第三种类型是布尔类型 <span class="Red-Font">**boolean**</span> ，有 true 和 false 两个值代表真和假。布尔类型主要在逻辑判断中去使用；
- 第四种类型是数字类型 <span class="Red-Font">**number**</span> ，它的值主要包含了整数和浮点数等数字，它还有一个特殊的值是 NaN 值。数字类型常常用来作为计算使用，NaN（直接读三个字母）是一个特殊的数字类型，它代表不是一个数字的意思（详见 JavaScript 第二题）；
- 第五种类型是字符串类型 <span class="Red-Font">**string**</span> ，它的值一般是被引号所包裹的文本字符串；
- 第六种和第七种是 ES6 新增的两种数据类型 <span class="Red-Font">**Symbol**</span> 类型和 <span class="Red-Font">**BigInt**</span> 类型，Symbol 代表创建后独一无二且不可改变的数据类型；BigInt 是一种数字类型的数据，它可以表示任意精度格式的整数。

引用数据类型只有1种，就是对象 <span class="Red-Font">**Object**</span> 类型（函数、数组等属于对象类型）

原始数据类型和引用数据类型的区别就在于原始数据类型的值是存放在栈中的，两个变量即使赋予相同的值，但他们的值也占据着不同的内存空间；而引用数据类型的值存放在堆中，如果两个变量的值相同的话，那他们的值存放在同一块内存空间中，两个变量的指针都指向这同一块内存空间。
:::
